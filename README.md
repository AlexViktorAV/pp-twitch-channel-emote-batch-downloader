# Twitch Channel Emote Batch Downloader

A lightweight Chrome extension that adds a button to Twitch channels to quickly download all emotes as a ZIP file.

## Features
- **One-Click Batch Download:** No need to save emotes individually.
- **Smart Fallback:** Automatically attempts to fetch animated versions (GIFs) and falls back to static versions (PNGs) if necessary.
- **Native Integration:** Injects a download button directly into the channel's settings menu.
- **Privacy Focused:** All processing happens locally in your browser. No data is stored or sent to external servers.

## Installation
1. Clone this repository or download the ZIP.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** in the top right.
4. Click **Load unpacked** and select this directory.

## Usage
1. Navigate to any Twitch channel.
2. Click the channel settings (three-dot menu).
3. Click the "Download [Channel Name] Emotes" button.
4. Your browser will download a ZIP file containing all available channel emotes.
